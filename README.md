# 🔍 PhishScan

Analisador de emails de phishing desenvolvido em Python.

## Funcionalidades
- Detecta palavras suspeitas no assunto e corpo do email
- Verifica domínios maliciosos
- Identifica links perigosos
- Calcula pontuação de risco (BAIXO, MEDIO, ALTO)
- Interface web para análise rápida
- Gera relatório automático

## Tecnologias
- Python 3.15
- Flask

## Como usar
1. Instale as dependências: `pip install flask`
2. Execute: `python src/app.py`
3. Acesse: `http://127.0.0.1:5000`
